# BBBS Data + Visualization Repo

This repository contains the data pipeline and dashboard outputs for geocoding addresses and visualizing route/location results.

## Quick Links

- [Latest data folder](data_geocode/latest/): Most recent geocoding output files (reports and datasets).
- [Latest geocoded CSV (`data_geocode.csv`)](data_geocode/latest/data_geocode.csv): Main row-level geocoded dataset for analysis or export.
- [Visualizations](https://stevenblum.github.io/bbbs/visualizations/dash_header.html): Overview and navigation page for the dashboards. Each visualization is a seperate HTML file here.
- [All-in-one Visualizations](https://stevenblum.github.io/bbbs/visualizations/dash_all_in_one.html) A single HTML file that combines all of the visualizations. Download this file if you want to use the visualizations offline.
